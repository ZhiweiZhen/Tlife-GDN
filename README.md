# Tlife-GDN

PyTorch implementation of Tlife-GDN: Detecting and Forecasting Spatio-Temporal Anomalies via Persistent Homology and Geometric Deep Learning, PAKDD 2022.

# Requirements:
Requirements \
Python >= 3.8 \
cuda == 11.1 \
Pytorch==1.11.0 \
PyG: torch-geometric==2.0.4
